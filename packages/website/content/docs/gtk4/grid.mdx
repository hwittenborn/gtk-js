## Behavior

- Children can specify `column`, `row`, `columnSpan`, and `rowSpan` props for explicit placement.
- Without placement props, children flow naturally.
- Uses CSS Grid for layout.

[GTK4 Documentation →](https://docs.gtk.org/gtk4/class.Grid.html)
