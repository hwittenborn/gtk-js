## Style Classes

| Class | Effect |
|-------|--------|
| `.horizontal` | Horizontal layout (default) |
| `.vertical` | Vertical layout |
| `.homogeneous` | Equal space for all children |
| `.linked` | Visually join children (e.g. button groups) |
