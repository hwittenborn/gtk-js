## Behavior

- Use `GtkNotebookPage` children to declare each tab and its content.
- Tabs can be positioned at top, bottom, left, or right via `tabPos`.

[GTK4 Documentation →](https://docs.gtk.org/gtk4/class.Notebook.html)
